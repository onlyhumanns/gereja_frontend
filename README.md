# 🕊️ Gereja Frontend - React + Vite

Frontend Sistem Informasi Gereja menggunakan React.js + Tailwind CSS.

## Cara Menjalankan

### 1. Install dependencies
```bash
npm install
```

### 2. Jalankan development server
```bash
npm run dev
```

Buka di browser: **http://localhost:5173**

## Catatan Penting
- Pastikan **backend sudah berjalan** di http://localhost:3000
- Login: `admin@gereja.com` / `admin123`
- Backend dan frontend harus dijalankan **bersamaan** di terminal berbeda
